Behavior and updates

  • Every list call is cached for about a week, I assume that the useage of this app isnt going to be heavy.
  • The cache is cleared everytime there is a new file uploaded or deleted, this keeps the list in check.
  • The source of truth is always your cloudflare dashboard, under R2 Onject storage, you will also see logs there.
  • Any updates from me to this app can be synced to yours from github. I will usually try my best to put a change log.
  • If you want the new updates click on Sync Fork, and it should trigger a auto deployment.
